## Features

### Modes

| Mode | Description |
|------|-------------|
| **Growth** | Standard compound interest with a fixed monthly contribution. Shows balance vs. total contributions over time. |
| **Burn to Zero** | Start from a lump sum and withdraw monthly — calculates when the account hits $0 (or whether interest outpaces withdrawals). |
| **Retire** | Two-phase simulation: contribute during an accumulation phase, then draw down in retirement. Shows the nest egg peak and the year funds run out. |
| **Waypoints** | Define contribution amounts that change at specific years (e.g., $50/mo until year 5, then $500/mo). Includes a phase breakdown table. |

### Other Features

- Live recalculation on every input change
- Adjustable compounding frequency (annually, quarterly, monthly, daily)
- Summary stats: final value, total contributed, interest earned, growth multiple
- Fully responsive — mobile-friendly with collapsible inputs panel

## Usage

No build step required. Just clone or download the repo and open `index.html` in any modern browser.

```bash
git clone https://github.com/tjsanzen/ComplexInterestCalculator.git
cd ComplexInterestCalculator
open index.html
```

All three files must be in the same directory:

```
index.html
style.css
script.js
```

Chart.js is loaded via CDN — an internet connection is required on first load.
